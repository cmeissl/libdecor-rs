pub fn is_floating(&self) -> bool

Return true if the window is floating.

A window is floating when it’s not maximized, tiled, fullscreen, or in any similar way with a fixed size and state. Note that this function uses the “applied” configuration. If this function is used in the ‘configure’ callback, the provided configuration has to be applied via commit first, before it will reflect the current window state from the provided configuration.
